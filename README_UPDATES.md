## Для разработчиков

### Что нужно знать

1. **Breaking Change:** `/api/token/refresh/` теперь требует `device_id`
2. **Новые endpoints:** `confirm` и `complete` для бронирований
3. **Новые поля:** `bookingStatus` в календаре, `countServices` и `summaryPrice` в организациях
4. **Изменения в правах:** CLIENT может создавать организации, услуги только для одобренных организаций
